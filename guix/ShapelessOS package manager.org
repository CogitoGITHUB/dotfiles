#+TITLE: ShapelessOS Package Manager - Org-Mode Based System

* Overview

The ShapelessOS Package Manager is an org-mode based system
for managing packages through literate documentation. It combines:
- Full documentation (LFS-style depth)
- Automatic Guix package generation
- Git-based source tracking with automatic hash computation
- Emacs integration for smooth workflow

* Architecture

#+BEGIN_SRC
Org-Mode Package Definition ( use tempo ) 
    ↓
Git Fetch + Hash Computation (automatic)
    ↓
Guix Package Generation (reproducible build)
    ↓
Build to /gnu/store (immutable)
    ↓
Package Available System-Wide
#+END_SRC


* Usage Guide

** Initial Setup

#+BEGIN_SRC emacs-lisp :tangle no
;; In your init.el
(add-to-list 'load-path "~/.emacs.d/")
(require 'shapeless-package-manager)
(shapeless-init)

;; Start automatic updates
(shapeless-start-auto-update)
#+END_SRC

** Creating a Package

1. Create org file in =~/.emacs.d/packages/=
2. Define package with properties
3. Add documentation
4. Add source blocks for each field
5. Save (auto-compiles)

** Keybindings

| Key       | Function                      |
|-----------+-------------------------------|
| C-c s c   | Compile all packages          |
| C-c s u   | Update package at point       |
| C-c s U   | Update all packages           |
| C-c s p   | Preview generated Guix code   |
| C-c s i   | Show package info             |
| C-c s b   | Build package with Guix       |
| C-c s I   | Install package               |

** Workflow

#+BEGIN_SRC
1. Edit package org file
2. Save (auto-compiles to Guix .scm)
3. C-c s p (preview generated code)
4. C-c s u (update git hash if needed)
5. C-c s b (build with Guix)
6. C-c s I (install to system)
#+END_SRC

** Package Inheritance

:PACKAGE: my-custom-nginx

;; Can reference standard nginx package

  (inherit nginx)

* Troubleshooting

** Common Issues

*** Git fetch fails
#+BEGIN_SRC
Error: Failed to fetch from git repository
#+END_SRC

*Solution:* Check git URL, network connection, authentication

*** Hash computation fails
#+BEGIN_SRC
Error: Could not compute hash
#+END_SRC

*Solution:* Ensure git repository is accessible, commit exists

*** Guix build fails
#+BEGIN_SRC
Error: Guix build failed
#+END_SRC

*Solution:* Check generated .scm file syntax, verify dependencies

** Parallel Updates

#+BEGIN_SRC emacs-lisp :tangle no
;; Update packages in parallel (experimental)
(defun shapeless-parallel-update ()
  "Update all packages in parallel."
  (interactive)
  (let ((packages (shapeless-plist-keys shapeless-packages)))
    (dolist (pkg packages)
      (make-thread
        (lambda ()
          (shapeless-update-package-hash (symbol-name pkg)))))))
#+END_SRC

** Selective Compilation

#+BEGIN_SRC emacs-lisp :tangle no
;; Compile only changed packages
(defun shapeless-compile-changed ()
  "Compile only packages that have been updated."
  (interactive)
  (let ((changed '()))
    (dolist (pkg (shapeless-plist-keys shapeless-packages))
      (when (plist-get (plist-get shapeless-packages pkg) :needs-rebuild)
        (push pkg changed)))
    (message "Compiling %d changed packages..." (length changed))
    (shapeless-compile-packages)))
#+END_SRC
