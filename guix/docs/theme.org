* Overview

The system uses a unified theme system with 3 backgrounds and red accents.

* Color Palettes

| Theme | Background | Primary Accent | Secondary Accent | Intensity |
|-------|------------|----------------|------------------|-----------|
| Day | #FFFFFF (white) | #8B0000 (dark red) | #A52A2A (light red) | Bright |
| Dusk | #808080 (grey) | #8B0000 (dark red) | #A52A2A (light red) | Medium |
| Night | #000000 (black) | #8B0000 (dark red) | #A52A2A (light red) | Dark |

* Apps Covered

All apps use the same red accent colors across themes:

| App | Category | Config Location |

| Nushell | Shell | home/config/nushell/theme.nu |
| Atuin | Shell history | home/config/atuin.toml |

| OpenCode | AI assistant | home/config/opencode/themes/white.json |
| Wezterm | Terminal emulator | home/config/wezterm/wezterm.lua |
