* Quick Start

** Build System

#+BEGIN_SRC bash
cd ~/.config/guix
sudo guix system reconfigure config.scm
#+END_SRC

** Or use the build script (with notifications)

#+BEGIN_SRC bash
cd ~/.config/guix
guix repl -- agents/scripts/sys-build-complete.scm
#+END_SRC

* DECLARATIVE SYSTEM PRINCIPLE (CRITICAL!)

This is a DECLARATIVE Guix system. EVERYTHING is defined in config.scm and related .scm files.

The entire system can be rebuilt from scratch using only:
  =guix system reconfigure ~/.config/guix/config.scm=

If something exists on the system but is NOT declared in Guix config, it breaks reproducibility!

** What MUST be declared:**

- ALL packages in home/packages/* and root/packages.scm
- ALL system services in root/services/*
- ALL kernel/boot/bootloader config in kernel/*
- ALL user accounts and permissions in root/users.scm
- ALL locale, timezone, keyboard settings
- ALL desktop environment (hyprland, wezterm, zellij)
- ALL shell integrations (nushell config, atuin, starship, zoxide)
- ALL user configurations that should persist

** Package + Service Organization (IMPORTANT!)

Each package file in home/packages/ MUST include its related service definition!

- home/packages/utils/keyd.scm → package + shepherd service
- home/packages/utils/kanata.scm → package + shepherd service

root/services/ is ONLY for services requiring root privileges (openssh, tailscale, docker, greetd).

** Nushell Configuration

Nushell config files are in Guix at home/config/nushell/ and get symlinked to ~/.config/nushell:

- nushell package: home/packages/shell/nushell.scm
- atuin: home/packages/shell/atuin.scm (loaded via source in config.nu)
- starship: home/packages/shell/starship.scm (loaded via source in config.nu)
- carapace: home/packages/shell/carapace.scm (generates completions in config.nu)
- zoxide: home/packages/shell/zoxide.scm (loaded in env.nu)

* Workflow

1. Create/modify .scm files in appropriate directory
2. Test with Guix REPL: =guix repl= then =,load "path.scm"=
3. Build package: =guix build -f path.scm=
4. Build system: =sudo guix system reconfigure config.scm=
5. Git push: =cd ~/.config/guix && git add -A && git commit -m "..." && git push=

* Key Services

** Keyboard (Dvorak)
- keyd: System-wide key remapping (TTY + Wayland)
- kanata: Wayland keyboard layers/macros/tap-dance
- Config: /home/aoeu/.config/kanata/kanata.kbd

** Idle/Sleep Prevention
- hypridle: Hyprland idle daemon (24hr timeout)
- elogind: Lid switch ignored, power/suspend/hibernate keys ignored
- System stays on with lid closed

** Tailscale
- VPN for remote access via Termux
- Topic: LiterativeOS VPN
