** Quick Commands

| Action                    | Command                              |
|---------------------------|-------------------------------------|
| Update Guix              | =guix pull=                        |
| Apply config             | =sudo guix system reconfigure ...= |
| Rollback                 | =guix system roll-back=            |
| List generations         | =guix system list-generations=     |
