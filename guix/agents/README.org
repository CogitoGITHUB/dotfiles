* Git Integration (CRITICAL!)

This entire system is under git version control:
- Location: =~/.config/guix/=
- Use =gr= to rebuild and auto-commit
- Commit regularly to maintain version history

* IMPORTANT: NO SUDO FOR READING FILES!

ALL files are world-readable - NEVER use sudo to read:
- /gnu/store/*, /tmp/*, /etc/*, /var/log/*

ONLY use sudo for: writes, service management, system ops

* IMPORTANT: Package + Service in SAME file!

Each package file must include its related service definition:
- home/packages/utils/keyd.scm → package + service
- home/packages/utils/kanata.scm → package + service

Only root-level services go in root/services/ (openssh, tailscale, docker, etc.)

* Usage

When working on this system:
1. *Read START.org first!* (at session start)
2. Check [[file:packages/management.org][packages/management.org]] for package operations
3. Check [[file:system/commands.org][system/commands.org]] for system operations
4. Use Nushell scripts in =agents/scripts/= for repetitive tasks
5. Main config is at ~/home/aoeu/.config/guix/config.scm
6. *ALWAYS ask to git push after changes!*

* IMPORTANT: Keep Documentation Updated

All agent files are in org-mode format. Key features:
