|-----------------|--------------------------------------|
| linux.scm      | Linux-libre kernel + uinput module  |

** Users

| File           | Purpose                                      |
|----------------|---------------------------------------------|
| users/users.scm | Aggregates all user accounts               |
| users/aoeu.scm | User account definition                    |

Example =users/aoeu.scm=:
#+BEGIN_SRC scheme
(user-account
  (name "aoeu")
  (group "users")
  (home-directory "/home/aoeu")
  (supplementary-groups '("wheel" "netdev" "audio" "video" "uinput" "keyd" "docker"))
  (shell (file-append nushell "/bin/nu")))
#+END_SRC
