* Module Templates — Infrastructure

** Template C — Loader file

#+BEGIN_SRC scheme
(define-module (core-system user-space root loaders <CATEGORY>)
  #:use-module (core-system user-space root <CATEGORY> <NAME1>)
  #:use-module (core-system user-space root <CATEGORY> <NAME2>)
  #:re-export (<NAME1> <NAME2>)
  #:export (root-<CATEGORY>-packages))

(define root-<CATEGORY>-packages
  (list <NAME1> <NAME2>))
#+END_SRC

Loader with services:
#+BEGIN_SRC scheme
(define-module (core-system user-space root loaders <CATEGORY>)
  #:use-module (gnu services)
  #:use-module (gnu services <SERVICE-MODULE>)
  #:use-module (core-system user-space root <CATEGORY> <NAME1>)
  #:re-export (<NAME1>)
  #:export (root-<CATEGORY>-packages
            root-<CATEGORY>-services))

(define root-<CATEGORY>-packages
  (list <NAME1>))

(define-public root-<CATEGORY>-services
  (list (service <service-type>)))
#+END_SRC

** Template D — root.scm additions (new loader only)

#+BEGIN_SRC scheme
;; 1. Add to #:use-module list:
#:use-module (core-system user-space root loaders <CATEGORY>)

;; 2. Append to root-system-packages:
(define-public root-system-packages
  (append ...
          root-<CATEGORY>-packages
          ...))

;; 3. If services:
(define-public root-system-services
  (append ...
          root-<CATEGORY>-services
          ...))
#+END_SRC

** Template E — Container module (current flat pattern)

#+BEGIN_SRC scheme
(define-module (systems containers <NAME>)
  #:use-module (gnu system)
  #:use-module (gnu services)
  #:use-module (gnu services docker)
  #:use-module (systems containers core-container)
  #:export (<NAME>-os <NAME>-service))

(define-public <NAME>-os
  (operating-system
    (inherit container-os)
    (host-name "guixos-<NAME>")
    (packages (append (list <PKG>)
                      (operating-system-packages container-os)))))

(define-public <NAME>-service
  (service oci-container-service-type
           (list (oci-container-configuration
                   (image (oci-image
                            (repository "guixos-<NAME>")
                            (tag "latest")
                            (value <NAME>-os)))
                   (network "host")
                   (auto-start? #t)
                   (respawn? #t)))))
#+END_SRC

** Template G — Custom shepherd service in same file as package

When a package needs a custom shepherd service, define both in the same file:

#+BEGIN_SRC scheme
(define-module (core-system user-space root <CATEGORY> <NAME>)
  #:use-module (guix packages)
  #:use-module (guix download)
  #:use-module (guix gexp)
  #:use-module (guix build-system trivial)
  #:use-module (gnu services)
  #:use-module (gnu services shepherd)
  #:use-module (gnu packages base)
  #:use-module (gnu packages compression)
  #:use-module ((guix licenses) #:prefix license:)
  #:export (<NAME> <NAME>-service-type))

(define-public <NAME>
  (package ...))

(define (<NAME>-shepherd-service config)
  (list (shepherd-service
          (provision '(<NAME>))
          (requirement '(user-processes))
          (start #~(lambda _ (fork+exec-command (list #$(file-append <NAME> "/bin/<NAME>d")))))
          (stop #~(make-kill-destructor)))))

(define-public <NAME>-service-type
  (service-type
    (name '<NAME>)
    (extensions (list (service-extension shepherd-root-service-type <NAME>-shepherd-service)))
    (default-value '())
    (description "Run <NAME> daemon")))
#+END_SRC
See networking/tailscale.scm for a working example.

** Template H — Multi-package aggregator submodule

When a submodule groups several related packages from the same (gnu packages ...) module:

#+BEGIN_SRC scheme
(define-module (core-system user-space root <CATEGORY> <NAME>)
  #:use-module (gnu packages <GNU-MODULE>)
  #:re-export (<PKG1> <PKG2> <PKG3>))
#+END_SRC
See shell/system-monitor.scm (btop, htop, ncdu, glances), shell/power.scm (upower, tlp, acpi),
shell/archive.scm (zip, unzip, 7zip, unrar-free, zstd, xz), networking/tools.scm (nmap, wireshark, iperf, iproute).

** Template I — Package alias

When you want to export a Guix package under a different name:

#+BEGIN_SRC scheme
(define-module (core-system user-space root <CATEGORY> <NAME>)
  #:use-module (gnu packages <GNU-MODULE>)
  #:export (<ALIAS>))

(define-public <ALIAS> <ORIGINAL-PACKAGE>)
#+END_SRC
See networking/tools.scm: `(define-public bind-dns isc-bind)`.
