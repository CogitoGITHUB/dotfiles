* LiterativeOS — Architecture

** Load Path
Set in guixos.scm:
  (add-to-load-path "/home/aoeu/.config/guix/Operating-System")
Every module path resolves relative to this root.

** Full Chain — guixos.scm to a single package

guixos.scm
  -> (core-system core-system)                    core-system/core-system.scm
       -> (core-system user-space root root)       core-system/user-space/root/root.scm
            -> (core-system user-space root loaders shell)
                 -> (core-system user-space root shell nushell)  re-exports: nushell
                 -> (core-system user-space root shell starship) re-exports: starship
                 exports: root-shell-packages
            -> (core-system user-space root loaders core)
            -> (core-system user-space root loaders networking)
            -> ... all other loaders

** guixos.scm

#+BEGIN_SRC scheme
(add-to-load-path "/home/aoeu/.config/guix/Operating-System")
(use-modules (core-system core-system))
os
#+END_SRC

** core-system.scm

#+BEGIN_SRC scheme
(define-module (core-system core-system)
  #:use-module (core-system kernel-space kernel-space)
  #:use-module (core-system user-space root root)
  #:export (os))

(define-public os
  (operating-system
    (packages root-system-packages)
    (services (append kernel-system-services root-system-services))))
#+END_SRC

** root.scm — how packages and services are assembled

#+BEGIN_SRC scheme
(define-module (core-system user-space root root)
  #:use-module (core-system user-space root loaders shell)
  #:use-module (core-system user-space root loaders core)
  ;; ... all loaders
  #:export (root-system-packages root-system-services))

(define-public root-system-packages
  (append root-shell-packages
          root-core-packages))

(define-public root-system-services
  (append
    (list (service dhcpcd-service-type)
          (service openssh-service-type))
    root-networking-services
    root-containers-services
    root-music-services
    %base-services))
#+END_SRC

** Loader pattern — real example: loaders/shell.scm

#+BEGIN_SRC scheme
(define-module (core-system user-space root loaders shell)
  #:use-module (core-system user-space root shell nushell)
  #:use-module (core-system user-space root shell starship)
  #:re-export (nushell starship)
  #:export (root-shell-packages))

(define root-shell-packages
  (list nushell starship))
#+END_SRC

** Submodule pattern — real example: shell/nushell.scm

#+BEGIN_SRC scheme
(define-module (core-system user-space root shell nushell)
  #:use-module (gnu packages nushell)
  #:re-export (nushell))
#+END_SRC
