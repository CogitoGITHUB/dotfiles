* LiterativeOS — Architecture

** Load Path
Set in guixos.scm:
  (add-to-load-path "/home/aoeu/.config/guix/Operating-System")
Every module path resolves relative to this root.

** Full Chain — guixos.scm to a single package

guixos.scm
  -> (core-system core-system)                    core-system/core-system.scm
       -> (core-system kernel-space linux)        kernel-space/linux.scm
       -> (core-system user-space root root)     user-space/root/root.scm
            -> (core-system user-space root loaders <category>)
                 -> (core-system user-space root <category> <name>)  re-exports: <name>

** No external channels
Never add external channels. Package definitions go directly into submodules.

** Kernel Space

kernel-space/linux.scm exports:
- kernel (corrupt-linux with regdb fix)
- kernel-initrd
- kernel-firmware
- kernel-arguments
- kernel-modules
