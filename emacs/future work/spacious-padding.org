* Tab-Bar Echo Area (Context Without Chrome)
:PROPERTIES:
:PACKAGE: tab-bar-echo-area
:STRAIGHT: (tab-bar-echo-area :type git :host github :repo "protesilaos/tab-bar-echo-area")
:END:

** Configuration :init:
#+begin_src emacs-lisp
;; Width of the echo area display
(setq tab-bar-echo-area-width 30)

;; What to show in the echo area
(setq tab-bar-echo-area-format
'(" "
tab-bar-echo-area-current-tab-index
"/"
tab-bar-echo-area-tab-count
" · "
tab-bar-echo-area-buffer-name))
#+end_src

** Enable Mode :config:
#+begin_src emacs-lisp
(require 'tab-bar-echo-area)

;; This mode installs all necessary hooks internally
(tab-bar-echo-area-mode 1)
#+end_src

** Visual Discipline (Optional) :config:
#+begin_src emacs-lisp
(custom-set-faces
'(tab-bar-echo-area
((t (:inherit shadow)))))
#+end_src
