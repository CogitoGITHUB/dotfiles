** Description
:PROPERTIES:
:tangle: no
:END:

God mode configuration for Emacs. Provides modal editing similar to Vim's normal mode.

Repository: [[https://github.com/emacsorphanage/god-mode][GitHub]]

Version: 2.18.0

** Configuration

#+begin_src emacs-lisp
;; ════════════════════════════════════════════════════════════
;; § GOD-MODE — CONFIGURATION
;; ════════════════════════════════════════════════════════════

;; Don't enable god-mode globally if using helix-mode
;; Instead, use it as an alternative modal system
;; (god-mode-all)

;; Update cursor appearance in god-mode
(defun my-god-mode-update-cursor ()
  "Update cursor appearance based on god-mode state."
  (setq cursor-type (if (or god-local-mode buffer-read-only)
                        'box
                      'bar)))

(add-hook 'post-command-hook #'my-god-mode-update-cursor)

#+end_src

** Keybindings

#+begin_src emacs-lisp
;; ════════════════════════════════════════════════════════════
;; § GOD-MODE — KEYBINDINGS
;; ════════════════════════════════════════════════════════════

;; Toggle god-mode
(global-set-key (kbd "<escape>") #'god-mode-all)
(global-set-key (kbd "C-c g") #'god-local-mode)

;; Exit god-mode with 'i' (insert mode)
(define-key god-local-mode-map (kbd "i") #'god-local-mode)

;; Convenient access to C-x and C-c prefixes
(define-key god-local-mode-map (kbd ".") #'repeat)
(define-key god-local-mode-map (kbd "[") #'backward-paragraph)
(define-key god-local-mode-map (kbd "]") #'forward-paragraph)

#+end_src
