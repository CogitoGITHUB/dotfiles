** Description
:PROPERTIES:
:tangle: no
:END:

Helix modal editing for Emacs. Brings Helix editor keybindings and selection-first workflow to Emacs.

Repository: [[https://github.com/mgmarlow/helix-mode][GitHub]]

Version: 0.1.0

Requires Emacs >= 29.1

:PROPERTIES:

** Configuration

#+begin_src emacs-lisp
;; ════════════════════════════════════════════════════════════
;; § HELIX — CONFIGURATION
;; ════════════════════════════════════════════════════════════

;; Enable helix-mode globally
(helix-mode)

;; Enable multiple-cursors integration
(when (featurep 'multiple-cursors)
  (helix-multiple-cursors-setup))

;; Optional: Enable "jj" as escape sequence
;; (helix-jj-setup 0.2)

;; Configure relative line numbers in normal mode
(add-hook 'helix-normal-mode-hook
          (lambda ()
            (setq display-line-numbers 'relative)))

(add-hook 'helix-insert-mode-hook
          (lambda ()
            (setq display-line-numbers t)))

#+end_src
