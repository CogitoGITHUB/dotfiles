* OPENCODE.EL — Emacs Interface to OpenCode
:PROPERTIES:
:PACKAGE:  emacs-opencode-el
:CATEGORY: ai
:VERSION:  0.0.1
:END:

** Overview
:PROPERTIES:
:tangle: no
:END:

opencode.el provides an Emacs interface to the OpenCode AI coding agent. It enables interactive sessions with OpenCode directly through Emacs, leveraging AI-powered code generation, refactoring, and analysis.

*Key Features:*
- Interactive OpenCode sessions in Emacs
- Multi-buffer session management
- Markdown rendering for responses
- Git integration (magit)
- Code generation and refactoring
- Codebase analysis
- Question answering about your code

** Usage
:PROPERTIES:
:tangle: no
:END:

Start an OpenCode session:

#+begin_src emacs-lisp
M-x opencode-new-session  ;; Create a new session
M-x opencode-send-query   ;; Send a question/request to OpenCode
#+end_src

**Setup Requirements:**

1. Ensure OpenCode CLI is installed and in PATH:
   #+begin_src bash
   which opencode  ;; Should show /path/to/opencode
   opencode --version
   #+end_src

2. Configure OpenCode authentication if needed

3. Start using OpenCode from Emacs
