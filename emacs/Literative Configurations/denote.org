* Denote
:PROPERTIES:
:PACKAGE: denote
:CATEGORY: system
:END:

Simple note-taking system with file-based organization and org-id support.
Integrates with org-supertag for hierarchical tagging and note discovery.

** Initialization
:PROPERTIES:
:tangle: init
:END:

#+BEGIN_SRC emacs-lisp
;; Ensure org-id is loaded
(require 'org-id)
#+END_SRC

** Configuration
:PROPERTIES:
:tangle: config
:END:

#+BEGIN_SRC emacs-lisp
;; Use org-id instead of date-based filenames
(setq denote-directory (expand-file-name "~/denote"))
(setq denote-file-type 'org)
(setq denote-infer-keywords t)
(setq denote-sort-keywords t)

;; Use org-id for filenames instead of dates
(setq denote-prompts '(title keywords))
(setq denote-date-prompt-use-org-read-date nil)

;; Org-id configuration for denote files
(setq org-id-link-to-org-use-id 'create-if-interactive-and-no-custom-id)

;; Auto-add org-id to new denote files
(add-hook 'denote-mode-hook
          (lambda ()
            (when (and (buffer-file-name)
                       (string-match-p (regexp-quote (denote-directory))
                                       (buffer-file-name)))
              (org-id-get-create))))

;; Enable denote-mode for org files in denote directory
(defun denote--auto-enable ()
  "Auto-enable denote-mode for org files in denote directory."
  (when (and (buffer-file-name)
             (string-match-p (regexp-quote (denote-directory))
                             (buffer-file-name))
             (eq major-mode 'org-mode))
    (denote-mode 1)))

(add-hook 'org-mode-hook #'denote--auto-enable)

;; Link formatting with org-id
(setq denote-link-format-org-with-post-title-emphasis t)
#+END_SRC

** Usage with org-supertag

Denote notes can be tagged with org-supertag for hierarchical organization:

#+BEGIN_SRC org
* Meeting Notes :meeting:project-x:
:PROPERTIES:
:ID: unique-org-id
:KEYWORDS: meeting project
:END:

Discussion about denote integration...
#+END_SRC

Each denote file:
- Uses org-id (UUID) as unique identifier
- Maintains keywords for search/filtering
- Can be tagged with org-supertag categories
- Integrates with literate-config-system packages
