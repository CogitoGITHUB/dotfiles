* NOTEBOOK-MODE — Notebook Mode
:PROPERTIES:
:PACKAGE:  notebook-mode
:STRAIGHT: (notebook-mode :type git :host github :repo "rougier/notebook-mode")
:AFTER:    org
:DEPENDS:  svg-tag-mode
:CATEGORY: utils
:VERSION:  latest
:END:

** Description
:PROPERTIES:
:tangle: no
:END:

Notebook mode configuration for Emacs.

GNU Emacs computational notebook using the powerful org-mode. Provides a clean and legible interface for literate programming, allowing users to execute code blocks, mix multiple programming languages, and export to HTML or PDF.

**Note**: This package works best with `svg-tag-mode` installed, which will be automatically installed via the DEPENDS property.

Repository: [[https://github.com/rougier/notebook-mode][GitHub]]

Version: latest
