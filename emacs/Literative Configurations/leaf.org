* LEAF — Declarative Elisp Configuration
:PROPERTIES:
:PACKAGE:  leaf
:CATEGORY: framework
:VERSION:  4.5.5

** Overview
:PROPERTIES:
:tangle: no
:END:

Leaf is the declarative configuration framework that powers the literate-config-system. It provides clean macro-based syntax for configuring packages.

** Initialization
:PROPERTIES:
:tangle: init
:END:
