* Org-Supertag
:PROPERTIES:
:PACKAGE: org-supertag
:END:

Hierarchical tagging system for Org-mode with powerful query and filtering capabilities.

** Init

#+BEGIN_SRC elisp :init
(add-to-list 'load-path 
             (expand-file-name "lisp/org-supertag" user-emacs-directory))
#+END_SRC

** Config

#+BEGIN_SRC elisp :config
;; org-supertag doesn't have a mode, but provides supertag- commands
;; Enable vault indicator if you want to show current vault
(when (fboundp 'supertag-vault-indicator-mode)
  (supertag-vault-indicator-mode 1))
(message "✓ org-supertag loaded and ready")
#+END_SRC
