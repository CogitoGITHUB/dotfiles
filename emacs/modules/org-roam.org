* Org-Roam
:PROPERTIES:
:PACKAGE: org-roam
:CATEGORY: system
:END:

Powerful note-taking and knowledge management system using org-mode.
Enables bidirectional links, backlinks, and visual graph representations.
Uses slug-based filenames with internal org-id for unique identification.

** Initialization
:PROPERTIES:
:tangle: init
:END:

#+BEGIN_SRC emacs-lisp
;; Ensure org-roam database is initialized
(require 'org-roam)
#+END_SRC

** Configuration
:PROPERTIES:
:tangle: config
:END:

#+BEGIN_SRC emacs-lisp
;; Configure org-roam
(setq org-roam-directory (expand-file-name "~/org-roam"))
(setq org-roam-db-location (expand-file-name "~/.emacs.d/org-roam.db"))

;; Create roam directory if it doesn't exist
(unless (file-exists-p org-roam-directory)
  (make-directory org-roam-directory t))

;; Initialize org-roam database
(org-roam-db-autosync-mode)

;; Org-roam capture templates with slug-based filenames
;; ${slug} creates readable filenames like "my-cool-note.org"
;; org-id is stored in :PROPERTIES: drawer for tracking/linking
(setq org-roam-capture-templates
      '(("d" "default" plain
         "%?"
         :target (file+head "${slug}.org"
                           "#+title: ${title}\n")
         :unnarrowed t)
        ("l" "literature" plain
         "Source: %^{Source}\n\n%?"
         :target (file+head "${slug}.org"
                           "#+title: ${title}\n#+filetags: literature\n")
         :unnarrowed t)
        ("p" "project" plain
         "Status: %^{Status|TODO|ACTIVE|DONE}\n\n%?"
         :target (file+head "${slug}.org"
                           "#+title: ${title}\n#+filetags: project\n")
         :unnarrowed t)
        ("i" "insight" plain
         "%?"
         :target (file+head "${slug}.org"
                           "#+title: ${title}\n#+filetags: insight\n")
         :unnarrowed t)))

;; Key bindings for org-roam
(leaf org-roam
  :bind (("C-c n l" . org-roam-buffer-toggle)
         ("C-c n f" . org-roam-node-find)
         ("C-c n i" . org-roam-node-insert)
         ("C-c n c" . org-roam-capture)))
#+END_SRC

** Usage Examples

Create a new note:
- ~C-c n c~ to capture a new note
- Choose template (default, literature, project, insight)
- Title "My Cool Note" becomes filename ~my-cool-note.org~
- Org-id automatically created in :PROPERTIES: drawer

Find and link existing notes:
- ~C-c n f~ to find a note by title
- ~C-c n i~ to insert a link to another note
- Backlinks automatically created via org-id

View backlinks:
- ~C-c n l~ to toggle the backlinks buffer
- Shows all notes that link to current note

** File Organization

Files are named by slug (title slugified):
- "My Cool Note" → ~my-cool-note.org~
- "Package: avy" → ~package-avy.org~
- "Design Decision XYZ" → ~design-decision-xyz.org~

Internal tracking via org-id:
- Each file has unique UUID in :PROPERTIES: drawer
- Org-roam uses ID for all links and backlinks
- Filename changes don't break anything

** Integration with LiterativeOS

Org-roam notes can reference package documentation:
- Create notes about package functionality
- Link to package files in Literative Configurations/
- Use org-supertag for hierarchical organization of notes
- Query knowledge graph for design decisions and architectural notes
