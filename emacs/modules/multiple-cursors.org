** Description
:PROPERTIES:
:tangle: no
:END:

Multiple cursors configuration for Emacs. Edit multiple locations simultaneously.

Version: 1.4.0

** Initialization
:PROPERTIES:
:tangle: init
:END:

#+begin_src emacs-lisp
;; ════════════════════════════════════════════════════════════
;; § MULTIPLE-CURSORS — INITIALIZATION
;; ════════════════════════════════════════════════════════════

(require 'multiple-cursors)

#+end_src

** Configuration

#+begin_src emacs-lisp
;; ════════════════════════════════════════════════════════════
;; § MULTIPLE-CURSORS — CONFIGURATION
;; ════════════════════════════════════════════════════════════

;; Store multiple-cursors lists in user directory
(setq mc/list-file (expand-file-name "mc-lists.el" user-emacs-directory))

;; Load mc/lists on startup
(when (file-exists-p mc/list-file)
  (load mc/list-file))

;; Always run all commands for all cursors
(setq mc/always-run-for-all t)

#+end_src
